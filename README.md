# Agent Secret Store Homebrew Tap

Install the `ass` CLI:

```bash
brew install agent-secret-store/tap/ass
```

Or tap first, then install:

```bash
brew tap agent-secret-store/tap
brew install ass
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `ass` | CLI for Agent Secret Store — the secure vault built for AI agents |
